Number Game:

This project is a simple interactive number guessing game designed to engage users while demonstrating basic programming concepts such as loops, conditionals, and user input handling.

The system begins by generating a random number within a specified range, typically between 1 and 100. The user is then prompted to enter their guess for the generated number. After each guess, the program compares the user's input with the actual number and provides feedback indicating whether the guess is correct, too high, or too low. This feedback helps guide the user toward the correct answer.

The game continues in a loop, allowing the user to keep guessing until they successfully identify the correct number. To make the game more challenging and structured, a limit is placed on the number of attempts a user can make. If the user fails to guess the number within the allowed attempts, the game ends and reveals the correct number.

Additionally, the program includes an option for multiple rounds, enabling users to play again without restarting the application. This enhances user engagement and replayability.

Overall, this project serves as an effective beginner-friendly exercise to understand control flow, random number generation, and interactive program design.
